Escenario: Universidad 

Consigna: Sistema de matrícula universitaria 
Implementar un sistema de inscripción a materias para estudiantes de una universidad, utilizando estructuras de datos. 

Requisitos: 
- Utilizar un conjunto para almacenar las materias ofrecidas por la universidad. 
- Utilizar un diccionario para almacenar la información de los estudiantes, incluyendo nombre, número de documento y materias inscriptas. 
- Utilizar una cola para gestionar los pedidos de inscripción de estudiantes, procesados en orden de llegada. 
- Permitir agregar y quitar materias del conjunto. 
- Incluir funciones para inscribir y desinscribir materias por parte de los estudiantes. 
- Implementar una interfaz básica por consola o menú para realizar estas operaciones.
